# sengwardenzahl
Tischtenniszählgerät
Hiermit können Mannschaften ihre Einzel und Doppel bei Punktspielen digital zählen. 
Durch das einfache Tippen auf einem Tablet, welches mit einem externen Bildschirm angeschlossen ist, kann man durch besondere Features dem TT-Schiedsrichter das Zählen leichter machen.
Es gibt Features, die man mit einem analogen Zählgerät nicht machen kann, beispielsweise die automatische Eingabe der Sätze, oder die Erfassung der Satzergebnisse. Außerdem sind Sicherheitssperren drinnen, die ein nicht korrektes Spielergebnis (z.B. 14:10) verhindern.
